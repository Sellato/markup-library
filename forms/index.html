Forms

This is a lightweight css library that allows you to build complex forms in very little time. It degrades relatively nicely to IE6, IE7 and IE8 and is tested in IE6, IE7, IE8, Firefox 2, Firefox 3, Safari, Opera and Chrome.

Known issues

  • Long labels, help texts and inline errors are clipped to preserve the grid structure
  • Elements with the alignFields class are only allowed to have one help text and one inline error
  • The position relative and padding on each overflow hidden paragraph are to ensure the outline isn't clipped
  • There's 3 kinds of float classes: alignBlocks, alignFields and alignButtons. Use them accordingly.
  • Inactive form elements aren't perfect in IE6 and IE7
  • Dropdown options are clipped in IE6, IE7 and IE8
  • First label in a horizontal and oneLiner combination, won't have the right width in older browsers if the firstChild class is missing.
